# elevenlabs-stt

A CLI that drives ElevenLabs' **web** Speech-to-Text feature
(`elevenlabs.io/app/speech-to-text`) from a script, using your own logged-in
account. It replays the web app's internal API — so it inherits the free
account's credit quota instead of using the paid official API.

> Not affiliated with ElevenLabs. Use your own account. The web internal API
> can change without notice.

## Install

```bash
pip install -r requirements.txt
playwright install chrome   # for the one-time login only
```

Optional: install `ffprobe` (from ffmpeg) for audio-duration pre-checks.

## Usage

### 1. One-time login

```bash
python stt.py login
```

Opens Chrome → log into ElevenLabs (email/password, Google, Apple, or SSO).
The script saves a Firebase **refresh token** to `session.json` (gitignored).
The 1-hour JWT is auto-refreshed on each run; re-run `login` only when the
refresh token itself expires (rare) or you get an auth error.

### 2. Transcribe

```bash
python stt.py transcribe audio.m4a                      # defaults → out.srt
python stt.py transcribe audio.m4a --lang eng --subs -o out.srt
python stt.py transcribe audio.m4a --vocab "Maj3r,V社" --format vtt -o out.vtt
```

Defaults (overridable by flags or `config.toml`):
- `tag_audio_events` on, `include_subtitles` on, `no_verbatim` off,
  `use_speaker_library` off, `language` auto-detect, `export_format` srt.

### Config

Copy `config.example.toml` → `config.toml` and edit to set persistent defaults
(vocab list, default language, export format, etc.). CLI flags override config.

### Languages

```bash
python stt.py list-languages
```

`--lang` accepts `auto`, a language name (e.g. `english`), or any ISO 639-3
code (e.g. `eng`, `zho`, `jpn`). Auto-detect is the default.

## Limits

- File size: hard 1000 MB (rejected before upload).
- Duration: warned if > 10 min (free-account soft limit) when `ffprobe` is
  installed; not hard-blocked.
- Credits: free tier ≈ 10,000 credits (~12 min of audio/month, ≈13.9 cr/s).
  If insufficient, the server rejects the upload and the CLI shows the error.
  `--show-cost` prints a pre-upload estimate.

## Files

- `stt.py` — the CLI + client.
- `config.example.toml` — documented defaults.
- `session.json` — generated by `login` (creds, gitignored).

## Self-check (offline)

```bash
python stt.py selfcheck
```
